wifi_get_opmode_default

Function:

get WiFi operating mode that saved in flash

Prototype:

    uint8 wifi_get_opmode_default (void)

Parameters:

null

Return:

  • WiFi working modes:

    • 0x01: station mode

    • 0x02: soft-AP mode

    • 0x03: station+soft-AP mode

user_interface.h

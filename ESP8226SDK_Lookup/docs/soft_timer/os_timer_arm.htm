os_timer_arm

Function:
    Enable a millisecond timer.

Prototype:
    os_timer_arm ( os_timer_t *ptimer, uint32_t milliseconds, bool repeat_flag )

Parameters:
    os_timer_t *ptimer : Timer structure
    uint32_t milliseconds : Timing, Unit: millisecond
    
        ‣ if called system_timer_reinit, the maximum value allowed to input is 0x41893

        ‣ if didn’t call system_timer_reinit, the maximum value allowed to input is 0xFFFFFFF

    bool repeat_flag : Whether the timer will be invoked repeatedly or not

Return:
    null

Include:
    osapi.h
    
Remark:
• For the same timer, os_timer_arm (or os_timer_arm_us) cannot be invoked repeatedly. os_timer_disarm should be invoked first.
• os_timer_setfn can only be invoked when the timer is not enabled,

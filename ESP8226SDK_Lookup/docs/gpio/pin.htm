PIN Related Macros

PIN_PULLUP_DIS(PIN_NAME)
    Disable pin pull up
PIN_PULLUP_EN(PIN_NAME)
    Enable pin pull up
PIN_FUNC_SELECT(PIN_NAME, FUNC)
    Select pin function
    
Include:
    eagle_soc.h

Example:
    PIN_FUNC_SELECT(PERIPHS_IO_MUX_MTDI_U, FUNC_GPIO12); // Use MTDI pin as GPIO12.

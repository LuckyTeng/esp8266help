system_get_vdd33

Function:
    Measure the power voltage of VDD3P3 pin 3 and 4, unit：1/1024 V
    
Note:
    • system_get_vdd33 can only be called when TOUT pin is suspended
    • The 107th byte in esp_init_data_default.bin（0〜～127byte）is named as “vdd33_const“ , when TOUT pin is suspended 
    vdd33_const must be set as 0xFF， that is 255

Prototype:
    uint16 system_get_vdd33(void)

Parameters:
    none
    
Return:
    power voltage of VDD33, unit：1/1024 V
    


Include:
    osapi.h

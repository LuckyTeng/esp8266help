system_init_done_cb

Function:
    Call this API in user_init to register a system-init-done callback.

Prototype:
    void system_init_done_cb(init_done_cb_t cb)

Parameters:
    init_done_cb_t cb : system-init-done callback

    bool repeat_flag : Whether the timer will be invoked repeatedly or not
    
Return:
    null
    
Note:
    wifi_station_scan has to be called after system init done and station enable.

Include:
    osapi.h
    
Remark:
• For the same timer, os_timer_arm (or os_timer_arm_us) cannot be invoked repeatedly. os_timer_disarm should be invoked first.
• os_timer_setfn can only be invoked when the timer is not enabled,

Example:
    void to_scan(void) { wifi_station_scan(NULL,scan_done); }
    void user_init(void) {
        wifi_set_opmode(STATION_MODE);
        system_init_done_cb(to_scan);
    }

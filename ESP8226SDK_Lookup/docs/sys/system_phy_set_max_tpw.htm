system_phy_set_max_tpw

Function:

Set maximum value of RF TX Power, unit ：0.25dBm

Prototype:

    void system_phy_set_max_tpw(uint8 max_tpw)

Parameters:

uint8 max_tpw : maximum value of RF Tx Power, unit : 0.25dBm, range [0, 82] it can be set refer to the 34th byte (target_power_qdb_0) of Espressif esp_init_data_default.bin(0〜～127byte)

Return:

None

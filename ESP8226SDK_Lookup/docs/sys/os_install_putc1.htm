os_install_putc1

Function:

Register print output function.

Prototype:

    void os_install_putc1(void(*p)(char c))

Parameters:

void(*p)(char c) - pointer of print function

Return:

none

Include:

osapi.h

Example:

os_install_putc1((void *)uart1_write_char) in uart_init will set os_printf to be output from UART 1, otherwise, os_printf default output from UART 0.
